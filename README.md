📊 Retail Sales Performance Analysis Dashboard | Power BI

An interactive Retail Sales Analytics Dashboard built using Microsoft Power BI to analyze business performance, customer behavior, product demand, and revenue trends.

The dashboard provides insights into important business KPIs such as Revenue, Transactions, Customers, Quantity Sold, and Order Cancellations, helping businesses understand performance across different regions, city tiers, and time periods.

📌 Short Description / Purpose

This project analyzes retail sales data to uncover insights about customer purchasing behavior, sales trends, and operational performance.

The dashboard allows users to monitor key business metrics and explore performance across months, zones, and city tiers, helping identify patterns and opportunities for improvement.

🛠 Tech Stack

The dashboard was built using the following tools:

• Power BI Desktop – Data visualization and dashboard development
• Power Query – Data cleaning and transformation
• DAX (Data Analysis Expressions) – Creating KPIs and calculated measures
• Data Modeling – Relationships between sales, product, and geographic tables



📂 Data Source

The dataset contains retail sales transaction data including product details, order information, and geographical data.

Tables used:

• Sales Raw – Transaction data including orders, quantities, and cancellations
• Product Master – Product categories and product groups
• City Tier Data – Metro, Tier 1 & 2, Tier 3 classification
• Pincode Geography – Geographic mapping for location-based analysis

⭐ Key Dashboard Features
KPI Overview

The dashboard highlights key business metrics:

• Total Revenue: 54.51M
• Total Quantity Sold: 83.37K
• Total Customers: 83.37K
• Total Transactions: 84.76K
• Total Cancellations: 24.43K

Sales Trend Analysis

Line charts analyze trends in:

• Monthly transactions
• Monthly customers
• Monthly quantity sold

This helps understand business growth and demand patterns.

Revenue Analysis

A time-series chart tracks revenue trends over time, helping identify fluctuations in sales performance.

Interactive Filters

The dashboard allows users to filter data by:

• Quarter
• City Tier (Metro, Tier 1 & 2, Tier 3)
• Zone (East, North, South, West)

🧩 Data Model

The project uses a relational data model connecting multiple datasets:

• Sales Data
• Product Information
• City Tier Classification
• Geographic Pincode Mapping

This enables cross-filtering and deeper business analysis.
